## :star:设置为256色 `高版本已默认`
`Preferences > Profiles > Terminal > Terminal Emulation > Report Terminal Type`
选择 **xterm-256color**

## :star:复制上一个会话目录
`Preferences > Profiles > Working Directory`
 - [x] Reuse previous session’s directory
